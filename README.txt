__descrizione__: Software per la manutenzione e per la customizzazione dello smartphone Xiaomi Mi A2 Lite, il programma e' strutturato in base
                 alla struttura delle directory. Per il corretto funzionamento del software rispettare le seguenti specifiche:
                 1. Avviare lo script install.sh per creare la stuttura di directory di cui il programma ha bisogno;
                 2. Caricare i pacchetti zip delle custom rom nella cartella ROMS
                 3. Caricare i pacchetti zip delle GApps nella cartella GAPPS
                 4. Caricare eventuali aggiornamenti del Disabler Encryption, di Magisk e del file .img di twrp nella cartella data con rispettivamenti i nomi:
                    4.a dfet.zip Per il Disabler
                    4.b magisk.zip Per il file di installazione di Magisk
                    4.c twrp-boot.img Per il file di avvio di TWRP
